**Clone the project and access the folder**

```bash
git clone https://github.com/DouglasANS/PokeApi.git 
```
&&
```bash
cd PokeApi
```

**Follow the steps below**

```bash
# Install the dependencies
$ npm install

# To finish, run the application
$ npm start

# Well done, project is started!
```
